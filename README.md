# Fake Profile Detection Using Machine Learning Including Cloud Computing

## 📌 Project Overview
This project is a Fake Profile Detection System developed using Machine Learning and Cloud Computing concepts. The application detects whether a profile is fake or genuine based on user data analysis.

The project is built using Python and Django framework with machine learning integration.

---

## 🚀 Features
- Fake profile detection
- Machine learning model integration
- User-friendly web interface
- Django backend support
- Static and media file handling
- Cloud deployment support

---

## 🛠️ Technologies Used
